本文指导你安装 iOS Demo。

1. Safari浏览器打开链接：https://testflight.apple.com/join/rSq3pLkr
2. 没有你按照TestFlight，会有这样的提示（如果已经安装，移步步骤6）
![avatar](step1-cn.PNG)
3. 点击**在App Store中查看**， 下载TestFlight
![avatar](step2-cn.PNG)
4. 下载完成TestFlight，打开TestFlight，点击**继续**
![avatar](step3-cn.PNG)
5. 回到步骤2，点击**开始测试**
6. 点击**接受**，点击**安装**
![avatar](step4-cn.PNG)
7. 点击**打开**
![avatar](step5-cn.PNG)
